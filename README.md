# Transparent-Bridge

本项目是基于Qt的透明网桥工作原理图形化演示程序。程序能够演示处转发表从初始为空到逐步增加的逆向学习过程，包括学习时的泛洪、学习后的转发等。



具体介绍都在Documents文件夹下的技术手册和用户使用说明中，非常详细。
